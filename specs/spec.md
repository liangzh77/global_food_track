# 食物全球史 - 项目规格说明

## 项目概述

**项目名称**: 食物全球史 (Global Food Track)

**项目类型**: 鸿蒙 (HarmonyOS) 移动应用 + Web 调试版本

**项目定位**: 一款以食物和农作物的历史传播为主题的知识型应用，帮助用户了解日常食物的起源地、传播路径和历史背景。

## 核心功能

### 1. 作物探索
- 按类别浏览作物：谷物、蔬菜、水果、豆类、香料、饮料作物、油料作物、糖料作物、坚果
- 查看作物详情：起源地、起源时间、传播路径、当前主产区
- 了解作物的别名和简介

### 2. 食物探索
- 按类别浏览食物：主食、菜肴、饮品、甜点、小吃、调味品、腌制食品
- 查看食物详情：起源地、主要食材、传播历史
- 关联作物：查看制作该食物所需的原料作物

### 3. 地区探索
- 按地理层级浏览：大洲 → 国家 → 地区
- 查看某地区起源的作物和食物
- 查看某地区的主产作物和流行食物

### 4. 搜索功能
- 全局搜索：支持搜索作物、食物、地点
- 支持别名搜索（如搜索"包谷"可找到"玉米"）

### 5. 传播路径可视化
- 展示作物/食物从起源地到各地的传播路线
- 显示传播时间和途径（如"丝绸之路"、"哥伦布航行"等）

## 目标用户

- 对历史文化感兴趣的普通用户
- 食品、农业相关从业者
- 学生和教育工作者
- 喜欢烹饪、关心食材来源的人群

## 技术栈

### 鸿蒙应用
- **开发语言**: ArkTS (eTS)
- **UI 框架**: ArkUI 声明式开发范式
- **构建工具**: hvigor

### Web 版本（调试用）
- **框架**: Vue 3
- **语言**: TypeScript
- **构建工具**: Vite
- **路由**: Vue Router 4

## 数据来源

应用数据为静态嵌入，存储在以下位置：
- 鸿蒙版: `entry/src/main/resources/rawfile/data/`
- 知识库源: `knowledge-base/`

数据包括：
- 作物数据（按类别分文件）
- 食物数据（按类别分文件）
- 地理位置数据（大洲、国家、地区）
- 关系索引数据（起源地索引、食材关系、传播路线）

- 页面切换流畅，无卡顿

- 鸿蒙版：支持 HarmonyOS 3.0+
- Web 版：支持主流现代浏览器

- 当前版本：仅支持中文
